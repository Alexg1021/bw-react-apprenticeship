Variables and Constants

  • Variables
    • A data container
    • Foundation for most programming language
  • var, let, const
    • var is the old way of declaring js variables
    • let is the current way of declaring a changing variable
    • const is for variables that will not change (constant)
  • Variable Naming
    • camelCasing > snake_casing
    • No spaces
    • only letters and digits
    • exceptions: $ _
    • Don't start with the #

Coding Challenge

  • Create a js-challenge-one.js and js-challenge-one.html file
  • Copy the template.html file to use as your base html code for js-challenge-one.html file
  • link your js-challenge-one.js file to your html template
  • Use the following two exercises to help with your js practice
    • Exercise 1: Welcome message for a user
      • Start by initializing your name in a string to a variable called fullName
      • Initialize a welcome message string to a const variable
      • Concatenate the two strings to form a new string
      • User the alert method to render the new string
    • Exercise 2: Arithmetic operators
      • Assign the numbers 1-6 to six different variables
      • Using arithmetic operators do the following
        • Sum up all six values and assign to a new variable and console.log() the variable
        • Sum up the first three values and multiply the total by the 4th value, then assign this to a new variable and console.log() the variable
